## API Docs
- Authorization [Get auth message](https://dev.predict.fun/get-auth-message-25326899e0.md): Get the message to sign to generate a JWT token.
- Authorization [Get JWT with valid signature](https://dev.predict.fun/get-jwt-with-valid-signature-25326900e0.md): Provide a signature and get back a valid JWT that allows order actions
- Categories [Get categories](https://dev.predict.fun/get-categories-25326910e0.md): :::info
- Categories [Get category by slug](https://dev.predict.fun/get-category-by-slug-25326911e0.md): Get category information and its markets by slug

- Markets [Get markets](https://dev.predict.fun/get-markets-25326905e0.md): Get information about markets
- Markets [Get market by ID](https://dev.predict.fun/get-market-by-id-25552989e0.md): Get detailed information about a specific market by its ID
- Markets [Get market statistics](https://dev.predict.fun/get-market-statistics-25326906e0.md): Get statistics about a specific market
- Markets [Get market last sale information](https://dev.predict.fun/get-market-last-sale-information-25326907e0.md): Get the last sale information for a specific market
- Markets [Get the orderbook for a market](https://dev.predict.fun/get-the-orderbook-for-a-market-25326908e0.md): :::info[]
- Markets [Get market timeseries](https://dev.predict.fun/get-market-timeseries-32534695e0.md): 
- Markets [Get latest market timeseries value](https://dev.predict.fun/get-latest-market-timeseries-value-32534696e0.md): 
- Orders [Get order by hash](https://dev.predict.fun/get-order-by-hash-25326901e0.md): Get the order information for one of your own orders by its hash
- Orders [Get orders](https://dev.predict.fun/get-orders-25326902e0.md): Get a list of your own orders
- Orders [Get order match events](https://dev.predict.fun/get-order-match-events-25663812e0.md): Get a paginated list of order match events with optional filtering by category, market, minimum value and signer. The results are sorted by `executedAt DESC`.
- Orders [Remove orders from the orderbook](https://dev.predict.fun/remove-orders-from-the-orderbook-25326904e0.md): Remove orders from the orderbook for the JWT signer. 
- Orders [Create an order](https://dev.predict.fun/create-an-order-32534694e0.md): Create a LIMIT or MARKET order for the authenticated user.
- Orders [Remove orders by hash](https://dev.predict.fun/remove-orders-by-hash-38139973e0.md): Remove orders from the orderbook by their hash for the authenticated signer (max 100).
- Accounts [Get connected account](https://dev.predict.fun/get-connected-account-25326917e0.md): Get information for the connected user account
- Accounts [Get account activity](https://dev.predict.fun/get-account-activity-32534697e0.md): Get a paginated list of trading activity including orders, matches,
- Accounts [Set a referral](https://dev.predict.fun/set-a-referral-25326918e0.md): Set a referral code if not already set
- Positions [Get positions](https://dev.predict.fun/get-positions-32675933e0.md): Get data about your positions.
- Positions [Get positions by address](https://dev.predict.fun/get-positions-by-address-32675934e0.md): Get positions for a specific wallet address.
- Search [Search categories and markets](https://dev.predict.fun/search-categories-and-markets-27399810e0.md): Full-text search across categories and markets. Returns separate arrays for each type, sorted by relevance.
- OAuth [Finalize a OAuth connection](https://dev.predict.fun/finalize-a-oauth-connection-25326912e0.md): [This endpoint is not available for general use]
- OAuth [Get the orders for a OAuth connection](https://dev.predict.fun/get-the-orders-for-a-oauth-connection-25326913e0.md): [This endpoint is not available for general use]
